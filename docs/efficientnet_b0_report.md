## Особенности EfficientNet-B0

1. **Компактность:** Самая маленькая модель в семействе EfficientNet
2. **Эффективность:** Оптимальное соотношение точность/скорость
3. **Mobile-friendly:** Подходит для мобильных устройств и edge-вычислений
4. **Compound scaling:** Использует составное масштабирование глубины, ширины и разрешения
